This folder is for project-specific type declarations.

Do NOT add a global components shim (e.g., global-components.d.ts) mapping component tags to .vue files.
Those shims make VS Code/TypeScript prefer the .d.ts file for navigation instead of the actual .vue source.

Use VS Code's "Go to Source Definition" (bound in this workspace to Cmd+Shift+F12) to jump into .vue files from templates.
