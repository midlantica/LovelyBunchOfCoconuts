# Meme Images

This folder contains all the meme images displayed on the website.

## How to Add a New Meme Image

1. Upload your image file to this folder
2. Use a descriptive filename (e.g., `funny-cat-meme.png`)
3. After uploading the image, create a matching metadata file in the `content/memes/` folder

## Image Guidelines

- **Formats**: PNG or JPG recommended
- **Size**: Aim for around 1080px wide
- **Filenames**: Use descriptive names with hyphens between words (e.g., `climate-change-meme.png`)
- **Quality**: Clear, readable images work best

## Important Note

For each image you upload here, you must create a corresponding metadata file in the `content/memes/` folder with the same name but a `.md` extension.

For example:

- Image: `public/memes/funny-cat-meme.png`
- Metadata: `content/memes/funny-cat-meme.md`

See the \_\_ReadMe.md in the `content/memes/` folder for instructions on creating the metadata file.
