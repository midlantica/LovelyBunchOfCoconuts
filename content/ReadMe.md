# WakeUpNPC Content Guide

A quick, friendly guide for adding content. Keep it simple; the app does the heavy lifting.

## TL;DR (60 seconds)

- Put new items in one of these folders: `claims/`, `quotes/`, `memes/`.
- Use the provided templates in `templates/` to create new files.
- Names: use short, lowercase, dash-or-underscore separated filenames. Avoid spaces.
- Files or folders that start with `_` are ignored by the app.
- The wall renders in a pattern: [claim | claim] → [quote] → [meme | meme] → [quote] → repeat.

## Help index

- Claims: ./claims/ReadMe.md
- Quotes: ./quotes/ReadMe.md
- Memes: ./memes/ReadMe.md

## Rules that matter

- Don’t rename or move the three top folders (`claims`, `quotes`, `memes`).
- Keep filenames stable once shared—URLs are derived from them and content.
- Keep content short and clean. The layout prioritizes visual rhythm over randomness.
- Underscored items (like `_drafts/` or `_readme.md`) are skipped automatically.

## Templates

- Claim: ../templates/claim.md
- Quote: ../templates/quote.md
- Meme: ../templates/meme.md

If unsure, copy a nearby file that looks right and tweak it.
